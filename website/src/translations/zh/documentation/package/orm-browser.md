# Deepkit ORM 浏览器

```sh
npm install @deepkit/orm-browser
```

Deepkit ORM 浏览器是一个 Web 应用程序，可用于浏览数据库 ORM 模式、编辑内容、查看迁移变更以及为数据库填充种子数据。

它是[框架调试器](../framework.md)的一部分，但也可以单独使用。
