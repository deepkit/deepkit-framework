# Deepkit ORM Browser

```sh
npm install @deepkit/orm-browser
```

Deepkit ORM Browser は、Database ORM schema を閲覧し、コンテンツを編集し、Migration の変更を確認し、データベースを seed できる Web アプリケーションです。

これは [Framework Debugger](../framework.md) の一部ですが、スタンドアロンでも利用できます。
